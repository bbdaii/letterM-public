# Letter M - Interactive 3D Project

一個基於 Three.js 的互動式 3D 字母 M 展示專案。

## 🎬 Demo

**線上展示:** https://bbdai.github.io/09-letter-m/

## ✨ 功能特色

- 🎯 **滑鼠互動**: Hover 字母 M 時旋轉加速
- 🎨 **材質切換**: 滾動滾輪切換 5 種不同材質
- ⭐ **背景粒子**: 視差效果的星星背景
- 💡 **動態光源**: 點光源跟隨滑鼠移動
- 🌟 **環境光照**: HDR 環境貼圖提供真實反射
- ✨ **自定義 Cursor**: 跟隨滑鼠的自定義游標

## 🛠️ 技術棧

- **Three.js** - 3D 渲染引擎
- **GSAP** - 動畫庫
- **Vite** - 建構工具
- **WebGL** - 圖形渲染

## 📦 安裝與執行

### 安裝依賴
```bash
npm install
```

### 開發模式
```bash
npm run dev
```

### 建構生產版本
```bash
npm run build
```

### 部署到 GitHub Pages
```bash
npm run deploy
```
